## contenido:
este repositorio contiene las verciones ya entrenadas de nuestro modelo kamutini y se planea que sea un lugar donde se documanten las verciones de el chatbot, el objetivo es que usen los resultados del entrenamiento y que ustedes puedan usar o modificar el modelo de manera en la que lo mejoren y adapten a sus nesecidades.

## objetivo: 
el objetivo de este modelo es ser open surse con los datos disponibles y listo para usar y modificar, esto ayuda a que ustedes puedan usar y/o modificar el modelo sin tener que crear un dataset desde cero limpiar los datos ets.
